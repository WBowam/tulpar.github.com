使用CSS3 Flexbox布局

  • Flexbox CSS3的属性
  • 用于简单快速的创建一个具有弹性功能的布局.

Felxbox的更新时间轴如下:

July 2009 Working Draft (display: box;)
March 2011 Working Draft (display: flexbox;)
November 2011 Working Draft (display: flexbox;)
March 2012 Working Draft (display: flexbox;)
June 2012 Working Draft (display: flex;)
September 2012 Candidate Recommendation (display: flex;)

标准的变动带来了,各个浏览器的兼容性问题.所以出现了全兼容代码一说,如下:

    display: -webkit-box; /* Chrome 4+, Safari 3.1, iOS Safari 3.2+ */
    display: -moz-box; /* Firefox 17- */
    display: -webkit-flex; /* Chrome 21+, Safari 6.1+, iOS Safari 7+, Opera 15/16 */
    display: -moz-flex; /* Firefox 18+ */
    display: -ms-flexbox; /* IE 10 */
    display: flex; /* Chrome 29+, Firefox 22+, IE 11+, Opera 12.1/17/18, Android 4.4+ */

至于判断哪些浏览器使用哪些代码,请参考浏览器引擎(WebKit及与之对应的浏览器引擎)

(以下内容都是使用2012年之后的标准)

概念

一个Flexbox布局是由一个伸缩容器（flex containers）和在这个容器里的伸缩项目（flex items）组成.

示例

1. 入门示例
<div class="container">
    <div class="item1">item1</div>
    <div class="item3">item1</div>
    <div class="item2">item1</div>
</div>

.container{
    height: 200px;
    display: flex;
    flex-flow: row;
    border: solid 1px;
}
.item1 {
    flex: 1;
    border: solid 1px;
}
.item2 {
    flex: 2;
    border: solid 1px;
}
.item3 {
    flex: 1;
    border: solid 1px;
}

效果如图: flex-flow: 指定了布局方向,默认为row,也可以设置column(竖直)

其他属性:
.container {
  display: flex;
  flex-flow: row wrap;
}

“wrap”，当伸缩项目在伸缩容器中无法在一行中显示的时候会另起一行排列。
