Git Diff

任意两个版本的差异

git diff master..test　　　

如果是比较当前工作目录与另一个分支，可以git diff lingyi-branch
默认情况下，当前工作目录和上次提交与本地索引间的差异

git diff

显示在当前的工作目录里的，没有 staged(添加到索引中)，且在下次提交(commit)时 不会被提交的修改

git diff --cached　

查看已经 staged(添加到索引中)，下次提交会提交的内容
已经staged的内容，用commit(不带 -a)提交到仓库
