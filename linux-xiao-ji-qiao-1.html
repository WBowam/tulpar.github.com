linux 小技巧（1）

1.install local deb packages:

dpkg -i file.deb

uninstall packages installed with dpkg:

``dpkg -r appname

list packages installed with dpkg:

dpkg --get-selections | grep -v deinstall

2.纯命令行下用鼠标

gpm

安装gpm服务

sudo apt-get install gpm

使用

serviece gpm start

3. -p参数用来创建多级目录

mkdir -p ./newdir/newdir2

4. 多线程下载工具　axel

axel -n 10 {url}
